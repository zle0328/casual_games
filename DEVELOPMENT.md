## 💡 技术决策

### 为什么选择 Cloudflare？

1. **零成本起步**: 免费额度够初期使用（日活 < 1000）
2. **无需服务器**: 全 Serverless，自动扩容
3. **全球 CDN**: 边缘计算，低延迟
4. **开发体验好**: Wrangler CLI 工具链完善

### 实时通信方案选择

**初期推荐: 轮询**
- 实现简单，完全免费
- 适合异步游戏（摇骰子、谁是卧底）
- 体验略差但可接受

**后期升级: Durable Objects**
- 真正实时（WebSocket）
- 需要 $5/月 付费计划
- 体验最佳

## 📝 注意事项

1. **先实现单机版**: 不依赖后端，快速看到效果
2. **渐进式开发**: 单机版 → 房间系统 → 联机版
3. **移动端优先**: 所有 UI 适配移动屏幕
4. **错误处理**: API 调用必须有异常处理和友好提示
5. **TypeScript**: 严格类型检查，减少运行时错误
